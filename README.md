# methodReverse

This is a function that takes in a word and returns the number of a's in the word.The function counts both lowercase 'a' and uppercase 'A'
## Built With

* Visual Studio Code
* HTML
* JavaScript

## Tested With
Live Server

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites:
* Code Editor of your choice

### To clone the repo to your local repository:

``` bash
git clone https://github.com/Kamogelo98/methodReverse.git
``` 

``` bash
cd methodReverse
```

Open ``` index.html ``` in your preferred browser
